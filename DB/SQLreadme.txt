===========================================
      HOW TO RUN THE PROJECT
===========================================

STEP 1: Copy the Project Folder
--------------------------------
Copy the "ConnectionSQL" project folder and paste it into:

C:\xampp\htdocs\

The folder structure should look like:

C:\
└── xampp
    └── htdocs
        └── ConnectionSQL
            ├── homepage.php
            ├── login.php
            ├── signup.php
            ├── logout.php
            ├── db.php
            └── ...

--------------------------------
STEP 2: Start XAMPP
--------------------------------
1. Open the XAMPP Control Panel.
2. Click "Start" for:
   ✔ Apache
   ✔ MySQL

Make sure both services are running (green).

--------------------------------
STEP 3: Open the Project
--------------------------------
Open your web browser and enter the following URL:

http://localhost/ConnectionSQL/homepage.php

You can also access the other pages directly:

Signup Page:
http://localhost/ConnectionSQL/signup.php

Login Page:
http://localhost/ConnectionSQL/login.php

Logout Page:
http://localhost/ConnectionSQL/logout.php

--------------------------------
STEP 4: Test the Project
--------------------------------
1. Register a new account using the Signup page.
2. Login using your registered credentials.
3. You will be redirected to the Homepage.
4. Click "Logout" to end your session.

===========================================
           PROJECT IS READY!
===========================================
